MorseBOX ADIF Exporter

Web tool for exporting logs from the MorseBOX program to ADIF format, which can be imported into popular amateur radio logging programs (e.g., Log4om, DXLog, etc.).

JavaScript HTML5 CSS3 ADIF

Features

  • Loading MorseBOX log files - Supports text files generated by MorseBOX version 7.02
  • Automatic log parsing - Extracts date, time, callsign, name (if provided – parsed from DR NAME if present) and RST report. Only the sent report is supported - the received report by default takes the same value and should be changed manually if it was different.
  • Duplicate filtering - Automatically removes duplicates of the same callsign within 2 minutes.
  • Data editing - Allows manual completion or correction of name, sent report, received report and band.
  • Export to ADIF - Generates a file in ADIF format with all necessary fields.
  • Deleting selected entries - Ability to remove unwanted entries before export.

How to use

  1. Enter your callsign in the "Your callsign" field
  2. Load the log file from MorseBOX (txt file)
  3. Check and correct the data in the table:
    • Complete names if they were incorrectly recognized
    • Check sent and received reports
    • Enter the band (in meters, e.g., 80, 20, 10...)
  4. Remove unwanted entries by selecting them and clicking "Delete selected"
  5. Click "Export to ADIF" to download the log.adi file
Note: The tool is not an official component of MorseBOX and is not formally affiliated with it.
However, it was developed in consultation with the author of MorseBOX, who approved its functional compatibility.

Requirements

  • Modern web browser with JavaScript enabled.
  • Log files from MorseBOX version 7.02

Supported ADIF fields

  • OPERATOR - Your callsign
  • CALL - Contacted station's callsign
  • QSO_DATE - QSO date (YYYYMMDD)
  • TIME_ON - Start time (HHMM)
  • MODE - Mode (always CW)
  • RST_SENT - Sent report
  • RST_RCVD - Received report
  • NAME - Name (if available)
  • BAND - Band (if provided)

MorseBOX ADIF Exporter

Narzędzie webowe do eksportowania logów z programu MorseBOX do formatu ADIF, które można zaimportować do popularnych programów do logowania łączności radiowych (np. Log4om, DXLog, itp.).

JavaScript HTML5 CSS3 ADIF

Funkcjonalności

  • Wczytywanie plików logów MorseBOX - Obsługuje pliki tekstowe wygenerowane przez MorseBOX w wersji 7.02
  • Automatyczne parsowanie logów - Wyodrębnia datę, czas, znak wywoławczy, imię (jeśli podane – odczytywane z DR NAME, jeśli występuje ) oraz raport RST. Obsługiwany jest wyłącznie raport nadany — raport odebrany domyślnie przyjmuje tę samą wartość i należy go zmienić ręcznie, jeśli był inny.
  • Filtrowanie powtórzeń - Automatycznie usuwa powtórzenia tego samego znaku wywoławczego w ciągu 2 minut.
  • Edycja danych - Umożliwia ręczne uzupełnienie lub poprawienie imienia, raportu nadawanego, raportu odbieranego oraz pasma.
  • Eksport do ADIF - Generuje plik w formacie ADIF z wszystkimi niezbędnymi polami.
  • Usuwanie zaznaczonych wpisów - Możliwość usunięcia niechcianych wpisów przed eksportem.

Jak używać

  1. Wpisz swój znak wywoławczy w polu "Twój znak"
  2. Wczytaj plik logu z MorseBOX (plik txt)
  3. Sprawdź i popraw dane w tabeli:
    • Uzupełnij imiona, jeśli zostały niepoprawnie rozpoznane
    • Sprawdź raporty nadawane i odbierane
    • Wpisz pasmo (w metrach, np. 80, 20, 10...
  4. Usuń niechciane wpisy zaznaczając je i klikając "Usuń zaznaczone"
  5. Kliknij "Eksportuj do ADIF" aby pobrać plik log.adi
Uwaga: Narzędzie nie stanowi oficjalnego komponentu MorseBOX i nie jest z nim formalnie powiązane.
Zostało jednak opracowane w konsultacji z autorem MorseBOX, który zatwierdził jego zgodność funkcjonalną.

Wymagania

  • Nowoczesna przeglądarka internetowa z włączoną obsługą JavaScript.
  • Pliki logów z MorseBOX w wersji 7.02

Obsługiwane pola ADIF

  • OPERATOR - Twój znak wywoławczy
  • CALL - Znak stacji pracującej
  • QSO_DATE - Data łączności (YYYYMMDD)
  • TIME_ON - Czas rozpoczęcia (HHMM)
  • MODE - Tryb (zawsze CW)
  • RST_SENT - Raport nadawany
  • RST_RCVD - Raport odbierany
  • NAME - Imię (jeśli dostępne)
  • BAND - Pasmo (jeśli podane)
